## Getting started
### Files
* server.c - C source code for server
* client.c - C source code for client

### Running the program
```
To build the server and client programs:
    make
```
```
To build the server program:
    make server
```
```
To build the client program:
    make client
```
```
To clean up the director:
    make clean
```
